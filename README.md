# Company Finder


## 🔈 Company Finder

Company Finder는 채용공고 정보를 리스트 형식으로 한눈에 보기 위해 학습 목적을 위해 만든 서비스 입니다.

해당 서비스는 모노레포로 구성이 되어 있으며 여러 기술들을 도입하며 개발 하고 있습니다.
